## Módulos do Lithe

O Lithe disponibiliza uma variedade de módulos criados pela equipe do Lithe, projetados para melhorar a funcionalidade e a eficiência da sua aplicação. Confira alguns dos principais módulos:

| Módulo | Descrição |
|--------|-------------|
| [Cache](/resources/modules/cache) | O módulo `Cache` do Lithe armazena e recupera dados frequentemente acessados, melhorando o desempenho do aplicativo com métodos para invalidação e recuperação com fallback. |
| [Crypt](/resources/modules/crypt) | O módulo `Crypt` utiliza AES-256-CBC para criptografar e descriptografar dados. Exige a variável `APP_KEY` e trata exceções durante a descriptografia. |
| [Hash](/resources/modules/hash) | O módulo `Hash` fornece métodos para criar e verificar hashes de dados sensíveis com Bcrypt, ideal para proteger senhas e dados críticos. |
| [Mail](/resources/modules/mail) | O módulo `Mail` simplifica o envio de e-mails, permitindo configurar servidores SMTP, enviar e-mails personalizados e adicionar anexos de forma prática. |

Para explorar mais módulos desenvolvidos pela equipe do Lithe, visite [Lithe Module](https://github.com/lithemod).
